## Draft Requirement Specification
ฉันต้องการทำโปรแกรมสำหรับใส่ลายน้ำลงใน PDF ของฉัน โดยมีคุณลักษณะดังนี้
1. พัฒนาด้วย Python และใช้ pattern การเขียนที่ minimal แต่ tough + powerfull แบบ MVP โดยใช้ bycdev-expert มาช่วยวาง pattern การเขียน
2. โปรแกรมจะรับ input arguments 2 ตัว คือ
    2.1. pdf full path
    2.2. ข้อความที่ใส่ใน pdf เพื่อพาดเป็นลายน้ำ
3. ขั้นตอนการ Process
    3.1. จะทำการแปลง PDF ที่ส่งเข้ามาให้เป็นภาพในแต่ละหน้า เพื่อป้องกันการ copy elements ออกไป หรือเอาไป edit ภายนอก
    3.2. สร้างข้อความลายน้ำจากข้อความที่ส่งเข้ามา โดยพาดในแต่ละหน้าของ PDF
    3.3. ตำแหน่งการวางลายน้ำให้เป็นตามมาตรฐาน ทั้งขนาด สี และจำนวนข้อความ โดยยังคงมองเห็นเนื้อหาเอกสารอยู่
4. Output นั้นต้องการเป็น pdf ที่มีชื่อไฟล์แบบเดียวกับ input file แต่ใส่ postfix ว่า _watermark.pdf
5. การพัฒนาโปรแกรมจะใช้ **BMAD MVP framework** (PRD > Architecuture > Milestone) โดยจะแยกไฟล์ออกจากกัน และไฟล์เหล่านั้นอยู่ใน folder @docs/
6. ในการพัฒนา ให้พัฒนาที่ละ Milestone และเมื่อทำกิจกรรมใดเสร็จ ให้ update เอกสารในแต่ละข้อของ Milestone ให้เป็น done
7. เลือกใช้ library ที่เหมาะสม ทำงานได้รวดเร็ว เป็นที่ยอมรับ และเสถียร
8. ใช้ไฟล์ pdf สำหรับทดสอบ ซี่งอยุ่ที่ @sample/test.pdf
